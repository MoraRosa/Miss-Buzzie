## üìä Org Chart

### **Planned Enterprise Features:**

- [ ] **Drag-to-reorder roles** within departments
  - Drag and drop to reorder roles in the same department
  - Visual feedback during drag operation
  - Auto-save after reordering

- [ ] **Zoom in/out controls** for large org charts
  - Zoom slider (50% - 200%)
  - Fit to screen button
  - Reset zoom button
  - Useful for organizations with 50+ employees

- [ ] **Collapse/expand departments** in preview mode
  - Click department to collapse/expand
  - Show employee count when collapsed
  - Remember collapsed state
  - Useful for large multi-department organizations

- [ ] **Contact information fields**
  - Email address
  - Phone number
  - Office location
  - LinkedIn profile
  - Display in preview and export

- [ ] **Export specific departments only**
  - Checkbox to select departments
  - Export only selected departments
  - Useful for sharing specific team structures

- [ ] **Export as CSV**
  - Export org chart data as spreadsheet
  - Columns: Name, Title, Department, Reports To, Email, Phone
  - Import into Excel, Google Sheets, etc.

- [ ] **Multi-level reporting chains**
  - Support for matrix organizations
  - Dotted line reporting relationships
  - Multiple managers per role

- [ ] **Department colors**
  - Assign custom colors to departments
  - Visual differentiation in preview
  - Color-coded exports
